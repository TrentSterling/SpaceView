SpaceView

See where your disk space goes. A fast, visual disk space analyzer inspired by SpaceMonger.

Download for Windows View Source
v0.5.1 — 3.6 MB standalone .exe

Features

▣

Treemap Visualization

Squarified treemap layout shows every file and folder as a proportionally-sized rectangle. Spot space hogs instantly.

↗

Smooth Navigation

Continuous zoom and pan with animated snap-to transitions. Scroll to zoom, double-click to dive in, right-click to zoom out.

⚡

Fast Scanning

Recursive directory scanner with real-time progress. See file count, total size, elapsed time, and scan rate as it runs.

🎨

Color Themes

Three built-in HSL color themes: Rainbow, Heatmap, and Pastel. Colors assigned by depth for instant visual hierarchy.

📦

Tiny & Portable

~3.6 MB standalone executable. No installer, no runtime dependencies. Just download and run.

🔗

Breadcrumb Trail

Always know where you are. Click any breadcrumb to jump to that directory level. Zoom indicator shows current magnification.

Controls

Scroll Zoom in / out at cursor
Double-click Snap zoom into folder
Right-click Zoom out to parent
Drag Pan the view
Backspace / Esc Zoom out to parent
Breadcrumbs Click to jump to any level

Install

Download

spaceview.exe

Grab the latest .exe from GitHub Releases. No installation needed.

Build from Source

git clone https://github.com/TrentSterling/SpaceView.git
cd SpaceView
cargo build --release

Requires Rust (edition 2021).
