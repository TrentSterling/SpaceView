SpaceView

See where your disk space goes. A fast, visual disk space analyzer inspired by SpaceMonger.

Download for Windows View Source
v0.6.0 - 3.6 MB standalone .exe

Scanning a 1.37 TB drive. Every rectangle is a file or folder, sized proportionally.

Built-in About dialog with keyboard shortcut reference.

Features

▣

Treemap Visualization

Every file and folder shown as a proportionally-sized rectangle. Big rectangles = big files.

↗

Smooth Navigation

Scroll to zoom, double-click to dive into folders, right-click to zoom out. Animated transitions between views.

⚡

Fast Scanning

Scans your drive with real-time progress. See file count, total size, elapsed time, and scan rate.

🎨

Color Themes

Three built-in color themes: Rainbow, Neon, and Ocean. Dark/light mode toggle. Colors assigned by depth for visual hierarchy.

📦

Tiny & Portable

~3.6 MB standalone .exe. No installer, no dependencies. Just download and run.

🔗

Breadcrumb Trail

Always know where you are. Click any breadcrumb to jump back. Zoom indicator shows current magnification.

Controls

Scroll Zoom in / out at cursor
Double-click Snap zoom into folder
Right-click Zoom out to parent
Drag Pan the view
Backspace / Esc Zoom out to parent
Breadcrumbs Click to jump to any level

Install

Download

spaceview.exe

Grab the latest .exe from GitHub Releases. No installation needed.

Build from Source

git clone https://github.com/TrentSterling/SpaceView.git
cd SpaceView
cargo build --release

Requires Rust (edition 2021).
